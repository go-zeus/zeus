# 贡献者行为准则

## 我们的承诺

为了营造一个开放友好的环境，作为贡献者和维护者，我们承诺使参与 Zeus 项目和社区的每个人都不受骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向身份。

我们承诺以促进开放、包容、多元化、健康社区的方式行动和互动。

## 我们的准则

营造积极环境的行为示例：

- 对他人展示同理心与善意
- 尊重不同意见、观点与经验
- 给予建设性反馈，也优雅地接受反馈
- 承担责任，向受我们错误影响的人道歉，并从中学习
- 关注的不仅是个人，更是整个社区的福祉

不可接受的行为示例：

- 性化语言或图像、性关注或挑逗
- 钓鱼、辱骂或贬损的评论；人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息（如物理或电子地址）
- 其他可被合理认定为不专业或在社区中不受欢迎的行为

## 维护者职责

项目维护者负责澄清和执行本行为准则的标准，并将在认为适当的情况下，对其认为不当、威胁、冒犯或有害的任何行为采取纠正措施。

项目维护者有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、issue 和其他贡献，并可暂时或永久禁止任何贡献者做出其他被视为敌对、骚扰或不可接受的行为。

## 适用范围

本行为准则适用于所有社区空间，在某人正式代表项目或其社区的公共空间中也适用。

代表项目的示例包括：使用官方项目电子邮件、通过官方社交媒体账户发布、或在线上或线下活动中担任指定代表。

## 执行

辱骂、骚扰或其他不可接受的行为可通过联系项目维护者团队 [GitHub Issues](https://github.com/go-zeus/zeus/issues) 私密举报（标记为 `conduct`）。所有投诉都将得到及时、公正的审查与调查。

所有社区成员都有义务尊重隐私与安全，并有责任报告违反本行为准则的行为。
